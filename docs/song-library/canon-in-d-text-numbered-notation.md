# Canon in D 卡农 文本简谱

来源说明：

- 标题：Canon per 3 Violini e Basso / Canon in D
- 作曲：Johann Pachelbel (1653-1706)
- 原作年代：1694
- 原始作品版权状态：公有领域。
- 转换来源：Mutopia Project `Canon_per_3_Violini_e_Basso` LilyPond 源文件，source 标注为 IMSLP。
- Mutopia 排版维护者：Michael Fischer v. Mollard。
- Mutopia 排版授权：Creative Commons Attribution 4.0。

转换说明：

- 本谱按 [文本简谱输入规范](../text-numbered-notation.md) 书写。
- 使用固定唱名 `1=C`，并用 `#` / `b` 表示临时升降号；因此 D 大调中的 `F#` 写作 `#4`，`C#` 写作 `#1`。
- 原编制的三把小提琴合并为右手 `RH`，同一时间起音的多个音使用 `[音+音]` 和弦写法。
- 原低音声部写入左手 `LH`，保留典型的 `D A B F# | G D G A` 循环。
- 当前文本简谱只有 `RH` / `LH` 两条时间线，不能表达三个独立右手声部的持续音重叠；本转写保留所有起音，并把相邻起音之间的间隔作为可播放时值。
- 这是完整 57 小节的可播放转写版，不对应任何现代商业钢琴编曲。
