# Greensleeves 绿袖子 文本简谱

来源说明：

- 标题：Greensleeves / 绿袖子
- 作曲：Traditional English folk song
- 早期来源：16 世纪英格兰民歌传统；同名 ballad 于 1580 年登记。
- 原始作品版权状态：公有领域。

转换说明：
